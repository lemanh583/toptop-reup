# Plan: 01-01 Core Downloader Infrastructure

## Objective
Xây dựng API Backend với FastAPI, hàng đợi Celery/Redis để tải video Tiktok (thông qua Github TikTokDownloader) không có watermark và lưu vào thư mục storage chung. Hỗ trợ WebSocket để real-time đẩy trạng thái tải cho Frontend.

## Tasks

### 1. Khởi tạo Docker và Requirements
<read_first>
- docker-compose.yml
- backend/requirements.txt
</read_first>
<action>
- Tạo `docker-compose.yml` định nghĩa service `redis` (image: redis:alpine, port: 6379:6379).
- Tạo `backend/requirements.txt` với `fastapi`, `uvicorn`, `celery`, `redis`, `websockets`, `pydantic-settings`.
</action>
<acceptance_criteria>
- `docker-compose.yml` có service `redis`.
- `requirements.txt` có đủ Celery và FastAPI.
</acceptance_criteria>

<read_first>

</read_first>
