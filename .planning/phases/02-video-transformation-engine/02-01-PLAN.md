# Plan: 02-01 Video Transformation Engine

## Objective
Xây dựng engine FFmpeg bọc bởi FastAPI và Celery để hỗ trợ lách bản quyền: MD5 padding, lật (mirror), nhiễu (noise) và đổi tốc độ video.

### 2. FFmpeg Transform Service
<read_first>
- N/A
</read_first>
<action>
Tạo file `backend/services/transform.py`. Class `VideoTransformer`:
Cung cấp hàm `apply_basic_transform(input_path, output_path)` gọi `ffmpeg-python` để:
1. `hflip` (mirror hình ảnh)
2. `setpts=1.05*PTS` (làm chậm 5%) hoặc `setpts=0.95*PTS` (tăng tốc 5%)
3. Lưu ra file output.
Cung cấp hàm `apply_md5_pad(output_path)` để mở file output chế độ `a` (append) và ghi thêm vài byte ngẫu nhiên.
</action>
<acceptance_criteria>
- Hàm `apply_basic_transform` có import và gọi `ffmpeg`.
</acceptance_criteria>

### 3. Celery Tasks for Transform
<read_first>
- backend/tasks.py
- backend/services/transform.py
</read_first>
<action>
Trong `backend/tasks.py`, import `VideoTransformer`. Định nghĩa `@celery_app.task(bind=True, name="transform_video_task")` nhận tham số `video_id`. 
Logic: Đọc `input_path` từ `storage/`, khởi tạo output_path, cập nhật state='PROGRESS', chạy phương thức transform, sau đó padding MD5, gửi state='SUCCESS'. Trả về `{ "transformed_file": ... }`.
</action>
<acceptance_criteria>
- `backend/tasks.py` có hàm `transform_video_task`.
</acceptance_criteria>
