# Plan: 04-01 Web Admin Dashboard (UI)

## Objective
Khởi tạo Vite React App, thiết lập TailwindCSS và update CORS cho logic FastAPI để tiếp nhận API call. Dựng UI Admin có form gởi request và lắng nghe Websocket theo dõi tiến độ Job.

## Tasks

### 1. Update Backend CORS
<read_first>
- backend/main.py
</read_first>
<action>
Thêm `CORSMiddleware` vào `main.py` với allow_origins=["*"] để tiện dụng khi Frontend gọi tới Port 8000.
</action>
<acceptance_criteria>
- `backend/main.py` chạy qua check Python mà không báo syntax lỗi và có import `CORSMiddleware`.
</acceptance_criteria>

### 2. Scaffold Frontend & Tailwind
<read_first>
- N/A
</read_first>
<action>
Dùng cờ non-interactive chạy `npx -y create-vite@latest frontend --template react`.
Cd vào `frontend`, chạy `npm install`, sau đó `npm install -D tailwindcss postcss autoprefixer`, và `npx tailwindcss init -p`.
Cập nhật file `tailwind.config.js` trỏ mảng content vào `"./src/**/*.{js,jsx,ts,tsx}"`.
Thay thế Code css mặc định của `index.css` bằng directives đuôi tailwind.
</action>
<acceptance_criteria>
- File gốc `package.json` có các dependencies React và Tailwind.
- `index.css` có `@tailwind base;`.
</acceptance_criteria>

### 3. Setup Axios & React UI
<read_first>
- frontend/src/App.jsx
</read_first>
<action>
Chạy `npm install axios lucide-react` trong folder `frontend`.
Cập nhật file `App.jsx` với cấu trúc React Function Component cơ bản. Dựng Form lấy input người dùng, Submit call axios `POST http://localhost:8000/api/download`.
Hứng task_id trả về, gắn kết nối `new WebSocket()`. Render tracking UI với TailwindCSS màu sắc rõ nét.
</action>
<acceptance_criteria>
- `App.jsx` không còn giữ logo Vite. Code Form React sạch đẹp, chuẩn responsive Tailwind.
</acceptance_criteria>
